# Operating System
This file features a chosen list of recommended books on operating systems.



### Books

01. [Operating Systems: Three Easy Pieces](https://pages.cs.wisc.edu/~remzi/OSTEP/) 
This book can also be ordered via [Amazon](https://www.amazon.com/Operating-Systems-Three-Easy-Pieces/dp/198508659X). 
- This book provides a clear and conceptual understanding of how operating systems work. It breaks down complex topics like processes, memory, and file systems into "three easy pieces".
