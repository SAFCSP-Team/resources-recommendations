# Architecture
This file contains a selected list of recommended materials on computer architecture.


### Books
01. [Assembly Programming and Computer Architecture for Software Engineers](https://www.amazon.com/Assembly-Programming-Computer-Architecture-Engineers/dp/1943153329)
- This book provides a practical introduction to Assembly programming and computer architecture, focusing on the "why" and "how" behind core concepts. 


02. [Assembly Language for x86 Processors (Kip Irvine)](https://www.amazon.sa/-/en/Assembly-Language-X86-Processors-Irvine/dp/0133769402)

- This book simplifies complex concepts with clear examples and exercises while providing practical insights into writing and debugging machine-level programs.
