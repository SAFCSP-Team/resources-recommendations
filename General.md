# General 
Here are general books recommended to prepare you for more advanced computer science concepts.



### Books

1. [But How Do It Know](https://www.amazon.sa/-/en/But-How-Know-Principles-Computers/dp/0615303765)
- This book covers binary logic, data representation, hardware components (CPU, memory, input/output), software fundamentals, and system architecture.


2. [Think Like A Programmer](https://www.amazon.sa/-/en/Think-Like-Programmer-Introduction-Creative/dp/1593274246)

- This book covers Key programming concepts like classes, pointers, recursion, and data structures.
